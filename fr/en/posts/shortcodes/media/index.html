Media Shortcodes

22-10-2020 1 min de lecture ShortcodeManualBilibiliTencentYoukuiQiyiNetease Music

A detailed description of media shortcodes, such as Bilibili, Tencent, Youku, iQiyi and Netease Music.

Bilibili

1{{< bilibili "video id" >}}

Tencent

1{{< tencentvideo "vid" >}}

Youku

1{{< youku "XNTQwMTgxMTE2" >}}

iQiyi

1{{< iqiyi "vid" "tvid" >}}

Netease Music

1{{< neteasemusic "id" [auto [type]] >}}
Parameter Description
auto The auto controls whether to autoplay. Boolean and optional, default to false.
type The type parameter is optional. Default to 2.
