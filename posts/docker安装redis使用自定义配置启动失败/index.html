Docker安装redis使用自定义配置，启动失败

2021-03-30 1 分钟阅读 后端dockerredis

@TOC

代码如下：

1docker run -p 6379:6379 --name redis-test 
2-v /usr/local/docker/redis/redis.conf:/etc/redis/redis.conf
3-v /usr/local/docker/redis/data:/data 
4-d redis redis-server /etc/redis/redis.conf 
5--appendonly yes

但是始终不行，后来查原因为【redis.conf】文件里面有一个配置设置错误

因为之前是手动安装的redis，设置了后台启动，所以使用docker进行自定义配置时就会失败，修改此处，删掉容器重新运行即可，此记。

1daemonize no
